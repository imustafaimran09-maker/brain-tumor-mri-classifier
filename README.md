
# Brain Tumor MRI Classifier 🧠

A deep learning model that classifies brain MRI scans into 4 categories:
- Glioma
- Meningioma
- Pituitary tumor
- No tumor

## Results
- Epoch 1: 86.54% accuracy
- Epoch 2: 94.77% accuracy
- Epoch 3: 96.40% accuracy

## Built With
- Python
- PyTorch
- ResNet18 (Transfer Learning)
- 7,153 MRI images from Kaggle
